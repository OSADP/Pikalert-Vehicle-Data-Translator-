Detailed Description

performs range check on headlights values and set corresponding qc flag in probe_message to true or false, if the check passes or fails, respectively lights:parkingLightsOn = 128; lights:fogLightOn = 64; lights:daytimeRunningLightsOn = 32; lights:automaticLightControlOn = 16; lights:rightTurnSignalOn = 8; lights:leftTurnSignalOn = 4; lights:highBeamHeadlightsOn = 2; lights:lowBeamHeadlightsOn = 1; lights:hazardSignalOn = 24; lights:allLightsOff = 0;
