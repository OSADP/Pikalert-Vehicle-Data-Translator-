Member Data Documentation

short vdt_probe_message::abs

The value from the anti-lock breaking element: 0 - notEquipped 1 - turned OFF 2 - turned ON 3 - Active/Engaged

short vdt_probe_message::air_temp

The value from the air temperature element minus 40 in Celsius.

float vdt_probe_message::air_temp2

The value from the second air temperature element minus 40 in Celsius.

short vdt_probe_message::bar_press

The value from the barometric pressure element multiplied by 2 plus 580 in hPa: Vehicle's Environmental Barometric Pressure hPa=INT*2+580 Range of 580 to 1090 hPa

short vdt_probe_message::brake_boost

The value from the brake.boost element: 0 - notEquipped 1 - turned OFF 2 - turned ON

short vdt_probe_message::brake_status

A combination of different bits being set:

– B'0000 "All Off" – bit0 (0) – B'1000 Right Rear Active – bit1 (1) – B'0100 Right Front Active – bit2 (2) – B'0010 Left Rear Active – bit3 (3) – B'0001 Left Front Active – B'1111 "All On"

float vdt_probe_message::hoz_accel_lat

The value from the horizontal lateral acceleration element multiplied by 0.01 in m/s^2

float vdt_probe_message::hoz_accel_long

The value from the horizontal longitudinal acceleration element multiplied by 0.01 in m/s^2

short vdt_probe_message::lights

A combination of the follwoing bit settings:

*bit0 (0) – B'1000-0000 parkingLightsOn *bit1 (1) – B'0100-0000 fogLightOn *bit2 (2) – B'0010-0000 daytimeRunningLightsOn *bit3 (3) – B'0001-0000 automaticLightControlOn *bit4 (4) – B'0000-1000 rightTurnSignalOn *bit5 (5) – B'0000-0100 leftTurnSignalOn *bit6 (6) – B'0000-0010 highBeamHeadlightsOn *bit7 (7) – B'0000-0001 lowBeamHeadlightsOn – B'0001-1000 hazardSignalOn – B'0000-0000 allLightsOff

double vdt_probe_message::obs_time

Observation time for the probe message in UNIX seconds It is calculated from full_pos_utc_time_year, ..., full_pos_utc_time_second.

vdt_point vdt_probe_message::point

Point location lat, long, elevation for a vdt_point will be being extracted from the fullPos.lat, fullPos.longitude and fullPos.elevation elements. The following conversion formulas are used: For lat/long: xml-values * 1/8,000,000 in degrees For elevation: xml-vlaue * 0.1 - 1000

double vdt_probe_message::rec_time

Reception time for probe message in UNIX seconds. It is calculated from directory/timestamped filename

short vdt_probe_message::stab

The value from the stability control element: *0 - notEquipped *1 - turned OFF *2 - turned ON *3 - Active/Engaged

float vdt_probe_message::steering_angle

The value from the steering angle element multiplied by 0.02 in degrees.

short vdt_probe_message::steering_rate

The value from the steering rate element multiplied by 3 in degree/second

short vdt_probe_message::trac

The value from the traction control element: *0 - notEquipped *1 - turned OFF *2 - turned ON *3 - Active/Engaged

short vdt_probe_message::wipers_front_status

The value from front wipers status: notEquipped (0) off (1) intermittent (2) low (3) high (4) washer (5) automaticPresent (255) – Auto wipper equipped

float vdt_probe_message::yaw

The value from the yaw element multiplied by 0.01 in degrees/second
